# MODELOS — Corpus de Referência OCEAN

Pasta para PDFs de relatórios interpretativos de personalidade (uso interno TeclaPonto).

## Fontes aceitas

- Relatórios interpretativos NEO/Hogan (traduzidos, uso interno)
- Artigos OCEAN em contexto organizacional
- Manuais públicos Costa & McCrae / IPIP

## Licença

PDFs proprietários (Hogan, NEO interpretive) são **uso interno apenas** — não redistribuir.
Itens do instrumento vêm do pool IPIP (domínio público).

## Fixtures mínimos

Se não houver PDFs, `extract_knowledge.py` gera corpus a partir de templates literatura.
